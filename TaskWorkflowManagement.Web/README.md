# TaskWorkflowManagement.Web

Angular frontend for TaskWorkFlowManagement.

## Development server

Start the backend first from `TaskWorkFlowManagement.Api`, then run:

```bash
npm install
npm start
```

Open `http://localhost:4200/`. The dev server proxies `/api` requests to `http://localhost:5095`.

## Building

To build the project run:

```bash
npm run build
```
